About The Contact App

This is a simple web app created with Django web framework as a Kisan Network Fullstack Assignment. You can see the list of contacts in the contact list page. You can check more about a contact by clicking on the detail button. On the contact profile you can send a randomly generated 6 digit OTP to that contact. This is possible beacause of Twilio third party application. You can Know more about Twilio app here.
